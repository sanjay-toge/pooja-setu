React Native + Expo app skeleton with UI pages for:
- Browse & search temples and poojas
- Book a pooja (dummy payment/confirmation)
- Live/recording player using **expo-video** (SDK 54 compatible)
- Temple dashboard (UI only)
- Notifications list
- Multi-tenant ready (dummy data)

## Run
1. Install Node.js LTS (use nvm).
2. In this folder: `npm i` then `npm start` (or `npx expo start -c`).
3. Press `i` for iOS Simulator, `a` for Android, or scan the QR in Expo Go.

> This project targets **Expo SDK 54** and **React Native 0.81**. We use `expo-video` instead of `expo-av` (removed in SDK 54).
